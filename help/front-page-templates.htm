Nós temos dois Modelos especiais para a primeira página, um para cada fase do projeto. Você deve defini-los manualmente, sempre que for conveniente.

Capa/Inscrições

Para ser usado durante a Fase de Inscrições. Ele mostra uma image estática—a Imagem Destacada desta página—relacionada ao tema definido para o projeto, e um pequeno texto introdutório—definido na Meta Box Texto do Destaque.

Capa/Divulgação

Para ser usado durante a Fase de Divulgação. Ele contém: um carrossel com os Filmes selecionados, um pequeno texto introdutório—definido na Meta Box Texto do Destaque—, um link para a página Mostra Itinerante e um link para a página de Notícias.

Tamanhos das imagens

As imagens para os destaques—nos dois modelos—seguem a proporção 16:9. Como o site é adatável pode ser visualizado em telas de tamanhos diferentes, a proporção é mais importante que as dimensões exatas. Por exemplo: em telas maiores a imagem da Capa/Inscrições mede 698x392, mas num iPhone 5S ela mede 506x284.
